# MindsetCRM

> Professional Mini CRM for Coaches, Consultants, and Mindset Professionals  
> Clean, aesthetic, responsive, and fully interactive UI built with TailwindCSS.

---

## Project Overview

MindsetCRM is a lightweight, responsive Customer Relationship Management (CRM) dashboard designed for mindset coaches, consultants, and small teams.  
It allows users to manage **clients**, **leads**, and **settings**, all in a **single-page application style** with smooth navigation.

---

## Features

### General
- Fully responsive design (desktop + mobile)  
- TailwindCSS-powered UI with **custom theme colors**  
- Interactive **sidebar navigation** with icons  
- Smooth **section toggling** (Dashboard, Clients, Leads, Settings)  
- Dark/Light theme support  

### Dashboard
- Overview of CRM activity  
- Clean, minimal, and modern layout  
- Placeholder for analytics, charts, or KPIs  

### Clients
- Client management table/cards  
- Profile details, quick actions (View/Edit)  
- Hover effects for professional UI feedback  

### Leads
- Search and filter functionality  
- Table with status badges (New, Contacted, Qualified, Lost)  
- Quick action buttons with hover micro-interactions  
- Responsive layout: table scroll on mobile  

### Settings
- User profile management (name, email, avatar)  
- Theme toggle (Light/Dark)  
- Notification preferences (email & push)  
- Security settings (password change, 2FA)  
- Responsive grid layout with premium cards
